#  XYZ Technologies – Corporate Website

A modern responsive company website built using **React**, **TailwindCSS**, and **React Router**.  
The website showcases services offered by XYZ Technologies, including Cloud Computing, Cyber Security, AI & ML, and Blockchain.  
It also contains pages like Home, About, Services, Contact and includes a global footer & navigation bar.

---

##  Features

- Fully Responsive UI (Mobile – Tablet – Desktop)
- Built with React Components
- React Router Navigation
- TailwindCSS Styling
- Sticky Top Navigation Bar
- Contact Form (Axios POST Request)
- Footer included across all pages
- Modern UI animations & hover effects

#  XYZ Technologies – Backend API

This is the backend server for the **XYZ Technologies Website**, handling contact form submissions through a secure REST API.

---
##  Tech Stack


 Node.js- Runtime environment
 Express.js- Backend server framework
 MongoDB / Mongoose | Database- for saving messages
 Axios (Frontend)- Sends data to backend 
 CORS-  Allows frontend <-> backend communication
